# LONO — Клініка репродуктології та жіночого здоров'я

## Про проєкт
Вебсайт клініки LONO у Львові, що спеціалізується на репродуктивній медицині та жіночому здоров'ї. Проєкт охоплює публічний сайт із каталогом послуг, пакетами, командою лікарів і онлайн-записом.

## Стек
- Чистий HTML5 / CSS3 / Vanilla JS (без фреймворків)
- Контент: Markdown-файли у `Stories/` як джерело правди
- Прототипи: `.html` файли для демонстрації UI/UX рішень
- Мова інтерфейсу: українська

## Правила для Claude
- Зберігай Ukrainian мову в усіх текстах, назвах і UI-контенті
- Stories/*.md — джерело правди для контенту; узгоджуй HTML з ними
- Не змінюй структуру `Stories/` без підтвердження
- Назви класів CSS: kebab-case, семантичні (`hero-title`, не `div1`)
- JS: без jQuery, без сторонніх бібліотек без необхідності
- Перевіряй валідність HTML через https://validator.w3.org перед фіналізацією
- SEO-метадані мають відповідати Stories/*.md (title, description, keywords)
- Адаптивний дизайн: mobile-first, breakpoints: 375px / 768px / 1200px

## Компонентний підхід
- **Один стиль = один клас**: всі візуально однакові елементи мають використовувати спільний CSS-клас (материнський компонент). Зміна стилю в одному місці має впливати на ВСІ екземпляри по всьому сайту.
- **Коричневі кнопки**: `.btn-gold` — єдиний клас для ВСІХ коричневих кнопок (в пакетах, на картках послуг, featured-card-tag, hero та ін.). Якщо потрібно змінити стиль коричневої кнопки — міняй `.btn-gold`, а не окремі елементи.
- **Сині кнопки**: `.btn-primary` — єдиний клас для всіх синіх кнопок.
- **Не дублюй стилі**: якщо елемент виглядає як кнопка — він має використовувати `.btn` + відповідний модифікатор, а не мати власні стилі, які повторюють ті самі значення.
- **Коли користувач каже "зміни кнопку"** — він має на увазі материнський клас, який вплине на всі кнопки цього типу по всьому сайту.

## Архітектурні рішення
- **Single-file prototypes**: кожен прототип — окремий `.html` файл із вбудованим CSS/JS для портативності
- **Stories як spec**: markdown-файли в `Stories/` описують мету, контент, SEO та структуру кожної сторінки — це ТЗ, не просто нотатки
- **4 аудиторні сегменти**: сайт орієнтований на 4 групи відвідувачів — пари що планують вагітність, жінки з гінекологічними проблемами, пацієнти ЕКЗ, партнери B2B
- **Brand voice**: довірливий, людяний, без медичного жаргону
