# Testing — LONO

## Валідація HTML
- Перевіряти через https://validator.w3.org перед фіналізацією будь-якого прототипу
- Нульова кількість помилок — обов'язково; попередження — оглянути та виправити якщо можливо

## Візуальна перевірка
- Тестувати на 3 breakpoints: 375px (мобільний), 768px (планшет), 1200px (десктоп)
- Перевіряти у Chrome та Firefox як мінімум
- Перевіряти читабельність тексту (контраст, розмір шрифту)

## Контент
- Кожна сторінка/прототип має відповідати своєму Stories/*.md файлу
- Перевіряти: наявність усіх блоків з специфікації, правильні SEO-метадані, CTA-кнопки

## Checklist перед комітом
- [ ] HTML валідний
- [ ] Всі тексти — українською
- [ ] SEO title/description відповідає Stories/*.md
- [ ] Адаптивність перевірено на 3 breakpoints
- [ ] Немає broken посилань
- [ ] CSS класи — kebab-case
- [ ] JS без зовнішніх залежностей
