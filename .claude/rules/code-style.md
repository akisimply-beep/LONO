# Code Style — LONO

## HTML
- Стандарт: HTML5, DOCTYPE обов'язковий, `lang="uk"`
- Відступи: 2 пробіли
- Семантичні теги: `<header>`, `<nav>`, `<main>`, `<section>`, `<article>`, `<footer>`
- Атрибути: у подвійних лапках, булеві без значення (`<input disabled>`)
- Зображення: завжди `alt` атрибут із описом українською
- Максимальна довжина рядка: 120 символів

## CSS
- Іменування класів: kebab-case, семантичне (`hero-title`, `card-cta`, не `div1`, `red-text`)
- Підхід: mobile-first — базові стилі для мобільного, потім `@media (min-width: 768px)` і `@media (min-width: 1200px)`
- Breakpoints: 375px / 768px / 1200px
- CSS-змінні: для кольорів та типографіки (`--color-primary`, `--font-heading`)
- Відступи: 2 пробіли
- Не використовувати `!important` без крайньої необхідності
- Коментарі: обов'язкові для нетривіальних блоків (`/* Hero section */`)

## JavaScript
- Без jQuery, без сторонніх бібліотек (якщо не обговорено окремо)
- ES6+: `const`/`let`, arrow functions, template literals
- Функції: іменовані, описові назви (`handleFormSubmit`, не `fn1`)
- Коментарі: для складної логіки
- Відступи: 2 пробіли

## Контент
- Мова: виключно українська у всіх текстах, alt-атрибутах, placeholder-ах
- Brand voice: довірливий, людяний, без медичного жаргону
