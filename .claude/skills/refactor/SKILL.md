# Refactor Skill — LONO

Викликається коли: HTML-файл більше 400 рядків, CSS-блок більше 100 рядків, або JS-функція більше 30 рядків.

## Що аналізувати

### HTML
- Чи є повторювані блоки, які можна винести як шаблон/компонент
- Чи правильна семантична структура (h1→h2→h3, без пропусків)
- Чи є зайва вкладеність `<div>` яку можна спростити

### CSS
- Чи є дубльовані значення кольорів/розмірів → замінити на CSS-змінні
- Чи можна об'єднати схожі селектори
- Чи дотримано mobile-first порядок

### JavaScript
- Чи є повторюваний код → виносити у функції
- Чи можна розбити великі функції на менші
- Чи є магічні числа → замінити на іменовані константи

## Процес
1. Покажи що і де треба рефакторити
2. Запропонуй конкретний план (що → як)
3. **Не виконуй рефактор без підтвердження користувача**
4. Після рефакторингу — перевір що зовнішній вигляд та поведінка не змінились
