Підготовка до публікації сайту LONO.

1. Перевір валідність усіх HTML-файлів (структура, закриті теги, кодування UTF-8)
2. Перевір що всі SEO-метадані заповнені (title, description, og:tags)
3. Перевір адаптивність — чи прописані стилі для 375px / 768px / 1200px
4. Перевір що всі тексти — українською
5. Перевір посилання — жодних broken links між сторінками
6. Покажи summary: список файлів готових до деплою та список файлів із проблемами
7. Запитай підтвердження перед фінальним кроком

Не виконуй деплой без явного підтвердження користувача.
