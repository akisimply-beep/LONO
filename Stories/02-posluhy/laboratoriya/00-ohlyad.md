### 2. Напрями лабораторних досліджень
Картки (4 штуки):
- **Загальноклінічні аналізи** → `/posluhy/laboratoriya/zahalni-analizy/`
- **Гормональний профіль** → `/posluhy/laboratoriya/hormony/`
- **Генетичні дослідження** → `/posluhy/laboratoriya/henetyka/`
- **Мікробіологія** → `/posluhy/laboratoriya/mikrobiolohiya/`

### 3. Переваги лабораторії LONO
Три блоки (іконки + текст):
- **Швидкість** — більшість аналізів готові протягом 1–3 робочих днів
- **Точність** — сучасне обладнання, контроль якості
- **Зручність** — результати в особистому кабінеті або за посиланням, без черг

### 4. Як здати аналізи?
- Запис не обов'язковий для простих аналізів (уточнюйте по телефону)
- Забір крові — вранці натщесерце (для більшості аналізів)
- Мазки, сеча — за вказівкою лікаря

### 5. Аналізи для пар перед ЕКЗ
